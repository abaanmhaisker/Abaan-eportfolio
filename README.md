# Abaan Mhaisker | Engineering E-Portfolio

Welcome to the source repository for my personal engineering portfolio! This site is designed to showcase my projects, hackathon participation, and software development skills in a highly professional, interactive format.

## Overview
This portfolio is built as a completely static front-end web application (HTML/CSS/Vanilla JS) that operates without a backend, making it incredibly fast and able to be statically hosted (e.g., on GitHub Pages).

## Features
- **Responsive Architecture**: Built utilizing modern CSS constraints (Flexbox/Grid) to natively adapt to mobile devices.
- **Glassmorphism Design**: High-end styling using a sophisticated Emerald and Slate dark mode palette.
- **Dynamic Projects Showcase**: Consolidates separate internal projects (like the *Student Enrollment System*) locally within the same repository structure to enable smooth local or GitHub routing.
- **Interactive Terminal**: Custom CSS keyframe-animated command-line prompts summarizing personal information without relying on heavy external libraries or personal photography.

## Working with the Repository Locally
To view the site, you do not need any package installation:
1. Clone or download this repository.
2. Open the `portfolio` directory in VS Code or your preferred IDE.
3. Start a local server:
   ```bash
   python -m http.server 8080
   ```
4. Or use the built-in VS Code "Live Server" extension.
